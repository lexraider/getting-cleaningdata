The analysis file consists of 5 sections.

Section 1 reads in the individual data text files and assemble them into a single data frame, data.

Section 2 subsets the data to include only the mean & std columns.

Section 3 translate the activity code into their respective labels.

Section 4 renamed the column labels with decriptive variable names (refer to CodeBook).

Section 5 summarize the data with the average value of each variable for each activity & each subject.
